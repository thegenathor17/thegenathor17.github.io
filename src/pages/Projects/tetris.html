Tetris

by Nahum :D

Well, I think a game like this doesn't need any introduction, Tetris itself is kinda popular, at least 2 people had bought it along since Alexey Pajitnov realeased it. The intention is to make a tetris-like game only using JavaScript, HTML and CSS. It's a simple project that won't take much longer than a few hours.

Plan

Firstly, I'm gonna make a set up, a new folder for it, the logo and rework the sidebar so the people can access to it, once it's done I will start with HTML making the basic structure that will be only a basic premade structure I usually use for articles like this but adding some modifications.

Secondly, I will start scripting all the basic logic for the game, it's important to code the basic mesh where the game will take place, all the input reading and basic pieces (using some 2d arrays to store all the data the game will need).

Thirdly, I'ma do all the CSS needed for this, it won't be much so I expect it to be a short phase.
